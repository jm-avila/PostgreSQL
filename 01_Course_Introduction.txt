Database Overview

What are Databases?
    Databases are systems that allow users to store and organize data. They are useful when dealing with large amounts of data.
Why use Databases?
    Data Integrity
    Can handle massive amounts of data
    Quickly combine different datasets
    Automate steps for re-use
    Can support data for website and applications

Typical users
    Databases have a wide variety of users!
    Analysts
        Marketing
        Business
        Sales
    Technical
        Data Scientist
        Software Engineers
        Web Developers
    Anyone needing to deail with data.

SQL
Structured Query Language is a programming language used to communicate with relational databases. It's a greate tool to make use of a variety of databases.
example:
    SELECT customer_id, first_name, last_name
    FROM sales
    ORDER BY first_name
